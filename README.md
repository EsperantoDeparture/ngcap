# ngcap
This package runs ng build --watch on a project, every time ng build --watch
detects changes, the next commands are executed:
npx cap sync
npx cap open electron
<!-- ngcapstop -->

# Usage
<!-- usage -->
```sh-session
USAGE
  $ ngcap [-d project-directory]
If the argument project-directory is not provided, the current working
directory is used.
```
